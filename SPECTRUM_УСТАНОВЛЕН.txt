ЧТО БЫЛО СДЕЛАНО:

ДОСТУПНЫЕ КОМАНДЫ:
------------------

1. Генерация документации:
   php artisan spectrum:generate
   
2. Генерация с оптимизацией:
   php artisan spectrum:generate:optimized
   
3. Режим реального времени (watch):
   php artisan spectrum:watch
   
4. Экспорт в Postman:
   php artisan spectrum:export:postman
   
5. Экспорт в Insomnia:
   php artisan spectrum:export:insomnia
   
6. Запуск mock сервера:
   php artisan spectrum:mock
   
7. Управление кешем:
   php artisan spectrum:cache

ПРОСМОТР ДОКУМЕНТАЦИИ:
----------------------

Способ 1: Через watch mode
   php artisan spectrum:watch
   Откроется браузер с документацией

Способ 2: Через файл
   Откройте: storage/app/spectrum/openapi.json
   Используйте Swagger Editor или Postman для просмотра

Способ 3: Создать HTML страницу
   Скопируйте openapi.json в public/
   Создайте HTML с Swagger UI

КОНФИГУРАЦИЯ:
-------------
Файл: config/spectrum.php

Основные настройки:
- title: "ApartmentHub API"
- version: "1.0.0"
- route_patterns: ['api/*', 'api/v1/*', 'api/v2/*']
- cache: включен
- response_detection: включен

НАЙДЕННЫЕ API МАРШРУТЫ:
-----------------------
Всего: 20 маршрутов
- Аутентификация (Auth)
- Отели (Hotels)
- Комнаты (Rooms)
- Бронирования (Bookings)

СЛЕДУЮЩИЕ ШАГИ:
---------------
1. Запустите: php artisan spectrum:watch
2. Или экспортируйте в Postman: php artisan spectrum:export:postman
3. Настройте теги и группы в config/spectrum.php (опционально)
