# 🏨 Система бронирования отелей

Полнофункциональная система бронирования отелей на Laravel 12, созданная на основе проекта [fezya011/booking](https://github.com/fezya011/booking).

## ✨ Возможности

### Для гостей (без регистрации):
- 🔍 Поиск и фильтрация отелей по городу, звездам, цене
- 🏨 Просмотр информации об отелях
- 🛏️ Просмотр доступных номеров
- ⭐ Чтение отзывов

### Для зарегистрированных пользователей:
- ✅ Все функции гостей
- 📅 Создание бронирований
- 📋 Просмотр истории бронирований
- ❌ Отмена бронирований (за 24 часа до заезда)

### Для администраторов:
- ✅ Все функции пользователей
- ➕ Создание и редактирование отелей
- 🛏️ Управление номерами
- 📊 Просмотр всех бронирований
- ⚙️ Управление удобствами

## 🗄️ База данных

### Основные таблицы:
- **users** - пользователи (роли: user, admin)
- **hotels** - отели (название, адрес, звезды, рейтинг)
- **rooms** - номера (тип, вместимость, цена)
- **bookings** - бронирования (даты, гости, оплата)
- **amenities** - удобства (Wi-Fi, парковка, бассейн)
- **reviews** - отзывы об отелях
- **hotel_amenities** - связь отелей и удобств
- **room_amenities** - связь номеров и удобств

## 🚀 Установка

### Требования:
- PHP 8.2+
- Composer
- MySQL
- Node.js & NPM

### Быстрая установка:

1. **Клонируйте репозиторий**
```bash
git clone <your-repo-url>
cd <project-folder>
```

2. **Установите зависимости**
```bash
composer install
npm install
```

3. **Настройте окружение**
```bash
copy .env.example .env
```

Откройте `.env` и настройте базу данных:
```env
DB_DATABASE=booking_system
DB_USERNAME=root
DB_PASSWORD=
```

4. **Создайте базу данных**
```sql
CREATE DATABASE booking_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

5. **Запустите миграции и сиды**
```bash
php artisan key:generate
php artisan migrate
php artisan db:seed --class=BookingSystemSeeder
```

6. **Скомпилируйте ассеты**
```bash
npm run dev
```

7. **Запустите сервер**
```bash
php artisan serve
```

Откройте http://localhost:8000

## 🎨 Функциональность

### Отели
- Список отелей с пагинацией
- Поиск по названию, городу

- Управление удобствами
- Просмотр всех бронирований

## 🐛 Решение проблем

### Ошибка подключения к БД
Проверьте настройки в `.env` и убедитесь, что база данных создана

### Ошибка "Class not found"
```bash
composer dump-autoload
```

### Ошибки с правами доступа (Linux/Mac)
```bash
chmod -R 775 storage bootstrap/cache
```

### Очистка кэша
```bash
php artisan cache:clear
php artisan config:clear
php artisan route:clear
php artisan view:clear
```
