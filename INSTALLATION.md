# Инструкция по установке системы бронирования отелей

## Требования
- PHP 8.2 или выше
- Composer
- MySQL или PostgreSQL
- Node.js и NPM

## Шаги установки

### 1. Установка зависимостей PHP
```bash
composer install
```

### 2. Установка зависимостей JavaScript
```bash
npm install
```

### 3. Настройка окружения
Скопируйте файл `.env.example` в `.env`:
```bash
copy .env.example .env
```

### 4. Настройка базы данных
Откройте файл `.env` и настройте подключение к базе данных:
```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=booking_system
DB_USERNAME=root
DB_PASSWORD=
```

### 5. Создайте базу данных
Создайте базу данных с именем `booking_system` в MySQL:
```sql
CREATE DATABASE booking_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 6. Генерация ключа приложения
```bash
php artisan key:generate
```

### 7. Запуск миграций
```bash
php artisan migrate
```

### 8. Заполнение тестовыми данными
```bash
php artisan db:seed --class=BookingSystemSeeder
```

Это создаст:
- Администратора: `admin@booking.com` / `password`
- Пользователя: `user@booking.com` / `password`
- 2 отеля с номерами
- Удобства (Wi-Fi, парковка, бассейн и т.д.)

### 9. Компиляция ассетов
```bash
npm run dev
```

Или для продакшена:
```bash
npm run build
```

### 10. Запуск сервера разработки
```bash
php artisan serve
```

Приложение будет доступно по адресу: http://localhost:8000

## Учетные данные для входа

### Администратор
- Email: `admin@booking.com`
- Пароль: `password`

### Обычный пользователь
- Email: `user@booking.com`
- Пароль: `password`

## Функциональность

### Для гостей (без авторизации):
- Просмотр списка отелей
- Поиск и фильтрация отелей
- Просмотр информации об отеле
- Просмотр номеров

### Для авторизованных пользователей:
- Все функции гостей
- Создание бронирований
- Просмотр своих бронирований
- Отмена бронирований

### Для администраторов:
- Все функции пользователей
- Создание и редактирование отелей
- Создание и редактирование номеров
- Управление бронированиями
- Просмотр всех бронирований

## Структура базы данных

### Основные таблицы:
- `users` - пользователи системы
- `hotels` - отели
- `rooms` - номера в отелях
- `bookings` - бронирования
- `amenities` - удобства
- `hotel_amenities` - связь отелей и удобств
- `room_amenities` - связь номеров и удобств
- `reviews` - отзывы об отелях


## Возможные проблемы

### Ошибка "Class 'Route' not found"
Убедитесь, что файл `routes/auth.php` существует или закомментируйте строку `require __DIR__.'/auth.php';` в `routes/web.php`

### Ошибка подключения к базе данных
Проверьте настройки в файле `.env` и убедитесь, что база данных создана

### Ошибки с правами доступа
На Linux/Mac может потребоваться установить права:
```bash
chmod -R 775 storage bootstrap/cache
```

## Следующие шаги

После установки вы можете:

2. Настроить внешний вид в файлах `resources/views`
3. Добавить загрузку изображений для отелей и номеров
4. Настроить отправку email-уведомлений
5. Интегрировать платежную систему
